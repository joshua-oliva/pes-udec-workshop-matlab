Este taller práctico introduce a los estudiantes en el uso de **MATLAB®** como herramienta esencial para la ingeniería. El enfoque principal es dominar la navegación del entorno, la manipulación de arreglos, procesamiento y presentación de datos, permitiendo automatizar análisis desde las bases de la programación.

---

## 🎯 Objetivos del Taller
Aprenderás:
* **Dominio del entorno:** Navegar la interfaz de MATLAB® y trabajar con Live Scripts.
* **Uso de Arreglos:** Generación manipulación de vectores y matrices.
* **Indexación de Datos:** Selección de elementos específicos de arreglos, mediante indexación manual, máscaras, o funciones utiles.
* **Extracción de Datos:** Extracción de datos externos del tipo `.csv`, junto con el desempaquetado de estos.
* **Estructuras Lógicas:** Estructuras `for`, `if-else`.
* **Presentación de Datos:** Creación y estilización de gráficas para la visualización de datos.
